# SurfWeb

This project has been migrated to **Vue 3 + Vite + TypeScript**.

## Development server

Run `npm run dev` and open the local URL printed by Vite.

## Build

Run `npm run build` to build the project. The build artifacts will be stored in `dist/`.

## Preview

Run `npm run preview` to preview the production build locally.

## Notes

The original Angular source has been preserved in `src-angular/` for reference during migration cleanup.
